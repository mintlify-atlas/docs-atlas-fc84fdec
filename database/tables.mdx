## Usuarios (Users)

The Usuarios table manages user authentication, authorization, and preferences.

### Table Structure

```sql
CREATE TABLE Usuarios (
    Id INT IDENTITY(1,1) PRIMARY KEY,
    Nombre NVARCHAR(100) NOT NULL,
    Apellido NVARCHAR(100) NOT NULL,
    Password NVARCHAR(255) NOT NULL,
    Email NVARCHAR(255) NOT NULL UNIQUE,
    Rol NVARCHAR(500) NOT NULL DEFAULT 'Operador',
    Estado NVARCHAR(20) NOT NULL DEFAULT 'Activo',
    Preferencias NVARCHAR(MAX) NULL,
    FechaCreacion DATETIME NOT NULL DEFAULT GETDATE(),
    FechaModificacion DATETIME NOT NULL DEFAULT GETDATE()
);
```

### Column Definitions

| Column | Type | Constraints | Description |
|--------|------|-------------|-------------|
| **Id** | INT | PRIMARY KEY, IDENTITY | Auto-incrementing user identifier |
| **Nombre** | NVARCHAR(100) | NOT NULL | User's first name |
| **Apellido** | NVARCHAR(100) | NOT NULL | User's last name |
| **Password** | NVARCHAR(255) | NOT NULL | Hashed password (bcrypt in application) |
| **Email** | NVARCHAR(255) | NOT NULL, UNIQUE | User email (login identifier) |
| **Rol** | NVARCHAR(500) | NOT NULL | Comma-separated roles list |
| **Estado** | NVARCHAR(20) | NOT NULL | 'Activo' or 'Inactivo' |
| **Preferencias** | NVARCHAR(MAX) | NULL | JSON string for UI preferences |
| **FechaCreacion** | DATETIME | NOT NULL | Record creation timestamp |
| **FechaModificacion** | DATETIME | NOT NULL | Last modification timestamp |

### Available Roles

- `Administrador` - Full system access
- `Visor` - Read-only dashboard access
- `Operador Totes` - Tote management operations
- `Operador Despacho` - Dispatch operations
- `Operador de Llenado de Totes` - Fill operations

### Indexes

```sql
-- Primary Key (automatic)
CREATE UNIQUE CLUSTERED INDEX PK_Usuarios ON Usuarios(Id);

-- Unique Email Index (automatic)
CREATE UNIQUE NONCLUSTERED INDEX UQ_Usuarios_Email ON Usuarios(Email);

-- Estado lookup optimization
CREATE NONCLUSTERED INDEX IX_Usuarios_Estado ON Usuarios(Estado);

-- Role filtering
CREATE NONCLUSTERED INDEX IX_Usuarios_Rol ON Usuarios(Rol);
```

## Totes (Containers)

The core table for tracking container inventory, status, and history.

### Table Structure

```sql
CREATE TABLE Totes (
    Id INT IDENTITY(1,1) PRIMARY KEY,
    Codigo NVARCHAR(50) NOT NULL UNIQUE,
    Estado NVARCHAR(30) NOT NULL DEFAULT 'TOTE VACIO',
    Ubicacion NVARCHAR(100) NULL,
    Cliente NVARCHAR(255) NULL,
    Usuario NVARCHAR(255) NULL,
    Producto NVARCHAR(255) NULL,
    Lote NVARCHAR(100) NULL,
    FechaDespacho DATETIME NULL,
    FechaRetorno DATETIME NULL,
    Alerta NVARCHAR(20) NULL,
    Peso DECIMAL(10,2) NULL,
    CodigoRetail NVARCHAR(50) NULL,
    UsuarioCreacion NVARCHAR(255) NULL,
    UsuarioModificacion NVARCHAR(255) NULL,
    FechaCreacion DATETIME NOT NULL DEFAULT GETDATE(),
    FechaModificacion DATETIME NOT NULL DEFAULT GETDATE(),
    Activo BIT NOT NULL DEFAULT 1,
    Observaciones NVARCHAR(MAX) NULL
);
```

### Column Definitions

| Column | Type | Constraints | Description |
|--------|------|-------------|-------------|
| **Id** | INT | PRIMARY KEY, IDENTITY | Tote identifier |
| **Codigo** | NVARCHAR(50) | NOT NULL, UNIQUE | Tote code (TOTE-0001) |
| **Estado** | NVARCHAR(30) | NOT NULL | Current status |
| **Ubicacion** | NVARCHAR(100) | NULL | Current location |
| **Cliente** | NVARCHAR(255) | NULL | Assigned client name |
| **Usuario** | NVARCHAR(255) | NULL | Assigned operator/user |
| **Producto** | NVARCHAR(255) | NULL | Product contained |
| **Lote** | NVARCHAR(100) | NULL | Batch/lot number |
| **FechaDespacho** | DATETIME | NULL | Dispatch date |
| **FechaRetorno** | DATETIME | NULL | Expected/actual return date |
| **Alerta** | NVARCHAR(20) | NULL | Alert level (Baja/Media/Alta/Crítica) |
| **Peso** | DECIMAL(10,2) | NULL | Weight in kilograms |
| **CodigoRetail** | NVARCHAR(50) | NULL | Retail/tracking code (COD-000001) |
| **UsuarioCreacion** | NVARCHAR(255) | NULL | User who created record |
| **UsuarioModificacion** | NVARCHAR(255) | NULL | Last user to modify |
| **FechaCreacion** | DATETIME | NOT NULL | Creation timestamp |
| **FechaModificacion** | DATETIME | NOT NULL | Last modification timestamp |
| **Activo** | BIT | NOT NULL | Soft delete flag (1=active) |
| **Observaciones** | NVARCHAR(MAX) | NULL | Additional notes |

### Estado Values

- `TOTE VACIO` - Empty container, available for use
- `CLIENTE` - Dispatched to client
- `EN MANTENCION` - Under maintenance
- `LAVADO` - In washing/cleaning
- `REPROCESO` - In reprocessing
- `DESECHO` - Marked for disposal

### Indexes

```sql
CREATE UNIQUE CLUSTERED INDEX PK_Totes ON Totes(Id);
CREATE UNIQUE NONCLUSTERED INDEX UQ_Totes_Codigo ON Totes(Codigo);
CREATE NONCLUSTERED INDEX IX_Totes_Estado ON Totes(Estado) INCLUDE (Activo);
CREATE NONCLUSTERED INDEX IX_Totes_Ubicacion ON Totes(Ubicacion);
CREATE NONCLUSTERED INDEX IX_Totes_Cliente ON Totes(Cliente);
CREATE NONCLUSTERED INDEX IX_Totes_Producto ON Totes(Producto);
CREATE NONCLUSTERED INDEX IX_Totes_FechaDespacho ON Totes(FechaDespacho);
CREATE NONCLUSTERED INDEX IX_Totes_FechaRetorno ON Totes(FechaRetorno);
CREATE NONCLUSTERED INDEX IX_Totes_Activo ON Totes(Activo);
CREATE NONCLUSTERED INDEX IX_Totes_CodigoRetail ON Totes(CodigoRetail);
CREATE NONCLUSTERED INDEX IX_Totes_Lote ON Totes(Lote);
```

### Example Query

```sql
-- Find all totes dispatched to a client that are overdue for return
SELECT Codigo, Cliente, FechaDespacho, FechaRetorno, 
       DATEDIFF(day, FechaRetorno, GETDATE()) as DiasVencidos
FROM Totes
WHERE Activo = 1 
  AND Estado = 'CLIENTE'
  AND FechaRetorno IS NOT NULL
  AND FechaRetorno < GETDATE()
ORDER BY DiasVencidos DESC;
```

## Llenados_Totes (Fill Operations)

Tracks all tote filling operations with weight measurements and RFID tracking.

### Table Structure

```sql
CREATE TABLE Llenados_Totes (
    Id INT IDENTITY(1,1) PRIMARY KEY,
    RFID_Tag NVARCHAR(100) NOT NULL,
    Peso_Bruto DECIMAL(10, 2) NOT NULL,
    Peso_Neto DECIMAL(10, 2) NOT NULL,
    Tara DECIMAL(10, 2) NOT NULL,
    Producto NVARCHAR(255),
    Lote NVARCHAR(100),
    Usuario NVARCHAR(100) NOT NULL,
    Nombre_Operador NVARCHAR(200),
    Balanza INT,
    Copias INT DEFAULT 1,
    Fecha_Registro DATETIME DEFAULT GETDATE(),
    CodigoBarra NVARCHAR(100) NULL,
    Sincronizado BIT DEFAULT 0
);
```

### Column Definitions

| Column | Type | Constraints | Description |
|--------|------|-------------|-------------|
| **Id** | INT | PRIMARY KEY, IDENTITY | Fill operation ID |
| **RFID_Tag** | NVARCHAR(100) | NOT NULL | RFID tag identifier |
| **Peso_Bruto** | DECIMAL(10,2) | NOT NULL | Gross weight (kg) |
| **Peso_Neto** | DECIMAL(10,2) | NOT NULL | Net weight (kg) |
| **Tara** | DECIMAL(10,2) | NOT NULL | Tare weight (kg) |
| **Producto** | NVARCHAR(255) | NULL | Product filled |
| **Lote** | NVARCHAR(100) | NULL | Batch number |
| **Usuario** | NVARCHAR(100) | NOT NULL | Operator username |
| **Nombre_Operador** | NVARCHAR(200) | NULL | Operator full name |
| **Balanza** | INT | NULL | Scale/balance ID |
| **Copias** | INT | DEFAULT 1 | Number of label copies |
| **Fecha_Registro** | DATETIME | DEFAULT GETDATE() | Registration timestamp |
| **CodigoBarra** | NVARCHAR(100) | NULL | Generated barcode |
| **Sincronizado** | BIT | DEFAULT 0 | Sync status with main Totes |

### Indexes

```sql
CREATE NONCLUSTERED INDEX IX_Llenados_RFID ON Llenados_Totes(RFID_Tag);
CREATE NONCLUSTERED INDEX IX_Llenados_FechaRegistro ON Llenados_Totes(Fecha_Registro);
CREATE NONCLUSTERED INDEX IX_Llenados_Usuario ON Llenados_Totes(Usuario);
CREATE NONCLUSTERED INDEX IX_Llenados_Producto ON Llenados_Totes(Producto);
CREATE NONCLUSTERED INDEX IX_Llenados_CodigoBarra ON Llenados_Totes(CodigoBarra);
```

## Eventos (Events/Audit Log)

Comprehensive audit trail for all system operations.

### Table Structure

```sql
CREATE TABLE Eventos (
    Id INT IDENTITY(1,1) PRIMARY KEY,
    ToteId INT NULL,
    TipEvento NVARCHAR(50) NOT NULL,
    Descripcion NVARCHAR(500) NOT NULL,
    Usuario NVARCHAR(255) NULL,
    FechaEvento DATETIME NOT NULL DEFAULT GETDATE(),
    DatosAdicionales NVARCHAR(MAX) NULL,
    IpAddress NVARCHAR(45) NULL,
    UserAgent NVARCHAR(500) NULL,
    Severidad NVARCHAR(20) NOT NULL DEFAULT 'Info',
    Modulo NVARCHAR(50) NULL,
    Accion NVARCHAR(100) NULL,
    ResultadoExitoso BIT NOT NULL DEFAULT 1,
    TiempoEjecucion INT NULL,
    SessionId NVARCHAR(100) NULL,
    FOREIGN KEY (ToteId) REFERENCES Totes(Id) ON DELETE SET NULL
);
```

### Column Definitions

| Column | Type | Constraints | Description |
|--------|------|-------------|-------------|
| **Id** | INT | PRIMARY KEY, IDENTITY | Event ID |
| **ToteId** | INT | FK to Totes, NULL | Related tote (optional) |
| **TipEvento** | NVARCHAR(50) | NOT NULL | Event type |
| **Descripcion** | NVARCHAR(500) | NOT NULL | Event description |
| **Usuario** | NVARCHAR(255) | NULL | User who triggered event |
| **FechaEvento** | DATETIME | NOT NULL | Event timestamp |
| **DatosAdicionales** | NVARCHAR(MAX) | NULL | JSON metadata |
| **IpAddress** | NVARCHAR(45) | NULL | Client IP address |
| **UserAgent** | NVARCHAR(500) | NULL | Browser user agent |
| **Severidad** | NVARCHAR(20) | NOT NULL | Severity level |
| **Modulo** | NVARCHAR(50) | NULL | System module |
| **Accion** | NVARCHAR(100) | NULL | Action performed |
| **ResultadoExitoso** | BIT | NOT NULL | Success flag |
| **TiempoEjecucion** | INT | NULL | Execution time (ms) |
| **SessionId** | NVARCHAR(100) | NULL | User session ID |

### Severidad Levels

- `Debug` - Debugging information
- `Info` - Informational messages
- `Warning` - Warning conditions
- `Error` - Error conditions
- `Critical` - Critical failures

### Indexes

```sql
CREATE NONCLUSTERED INDEX IX_Eventos_ToteId ON Eventos(ToteId);
CREATE NONCLUSTERED INDEX IX_Eventos_FechaEvento ON Eventos(FechaEvento DESC);
CREATE NONCLUSTERED INDEX IX_Eventos_Usuario ON Eventos(Usuario);
CREATE NONCLUSTERED INDEX IX_Eventos_TipEvento ON Eventos(TipEvento);
CREATE NONCLUSTERED INDEX IX_Eventos_Severidad ON Eventos(Severidad);
CREATE NONCLUSTERED INDEX IX_Eventos_Modulo ON Eventos(Modulo);
CREATE NONCLUSTERED INDEX IX_Eventos_SessionId ON Eventos(SessionId);
CREATE NONCLUSTERED INDEX IX_Eventos_ResultadoExitoso ON Eventos(ResultadoExitoso);
CREATE NONCLUSTERED INDEX IX_Eventos_Accion ON Eventos(Accion);
```

## Ubicaciones (Locations)

Warehouse locations and zones for tote storage.

### Table Structure

```sql
CREATE TABLE Ubicaciones (
    Id INT IDENTITY(1,1) PRIMARY KEY,
    Nombre NVARCHAR(100) NOT NULL,
    Descripcion NVARCHAR(500) NULL,
    Codigo NVARCHAR(50) NULL,
    Tipo NVARCHAR(50) NULL,
    Area NVARCHAR(50) DEFAULT 'General',
    Capacidad INT NULL,
    ClientesAsignados NVARCHAR(MAX) NULL,
    Estado NVARCHAR(20) NOT NULL DEFAULT 'Activo',
    FechaCreacion DATETIME DEFAULT GETDATE(),
    FechaModificacion DATETIME DEFAULT GETDATE()
);
```

### Column Definitions

| Column | Type | Constraints | Description |
|--------|------|-------------|-------------|
| **Id** | INT | PRIMARY KEY, IDENTITY | Location ID |
| **Nombre** | NVARCHAR(100) | NOT NULL | Location name |
| **Descripcion** | NVARCHAR(500) | NULL | Location description |
| **Codigo** | NVARCHAR(50) | NULL | Location code (ALM-001) |
| **Tipo** | NVARCHAR(50) | NULL | Type classification |
| **Area** | NVARCHAR(50) | DEFAULT 'General' | Area classification |
| **Capacidad** | INT | NULL | Maximum capacity |
| **ClientesAsignados** | NVARCHAR(MAX) | NULL | JSON list of assigned clients |
| **Estado** | NVARCHAR(20) | NOT NULL | 'Activo' or 'Inactivo' |
| **FechaCreacion** | DATETIME | DEFAULT GETDATE() | Creation timestamp |

### Tipo Values

- `Almacén` - Warehouse/Storage

- `General` - General purpose
