## Product Admin Actions

Administrative CRUD operations for product management. All product endpoints require admin authentication.

```http
POST /api/admin/productos
```

### Authentication

All requests must include a valid JWT token in the Authorization header:

```
Authorization: Bearer YOUR_ADMIN_TOKEN
```

### Request Body

<ParamField body="action" type="string" required>
  Action to perform: "create", "list", "update", or "delete"
</ParamField>

<ParamField body="productData" type="object">
  Product data object (required for create, update, and delete actions)
</ParamField>

---

## List Products

Retrieve all products in the catalog.

### Request Body

<ParamField body="action" type="string" required>
  Set to "list"
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="productos" type="array" required>
  Array of product objects
  
  <ResponseField name="Id" type="integer">
    Unique product identifier
  </ResponseField>
  
  <ResponseField name="Nombre" type="string">
    Product name
  </ResponseField>
  
  <ResponseField name="Descripcion" type="string">
    Product description
  </ResponseField>
  
  <ResponseField name="Codigo" type="string">
    Product code/SKU
  </ResponseField>
  
  <ResponseField name="Categoria" type="string">
    Product category
  </ResponseField>
  
  <ResponseField name="Estado" type="string">
    Product status (e.g., "Activo", "Inactivo")
  </ResponseField>
  
  <ResponseField name="Ingredientes" type="string">
    List of ingredients
  </ResponseField>
  
  <ResponseField name="Clientes" type="string">
    Assigned clients for this product
  </ResponseField>
  
  <ResponseField name="FechaVencimiento" type="integer">
    Shelf life in days
  </ResponseField>
</ResponseField>

### Example Request

```bash
curl -X POST "https://api.ditzlertotes.com/api/admin/productos" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "action": "list"
  }'
```

## Create Product

Create a new product in the catalog.

### Request Body

<ParamField body="action" type="string" required>
  Set to "create"
</ParamField>

<ParamField body="productData" type="object" required>
  Product information object
</ParamField>

### ProductData Fields

<ParamField body="productData.nombre" type="string" required>
  Product name (must be unique)
</ParamField>

<ParamField body="productData.descripcion" type="string">
  Product description
</ParamField>

<ParamField body="productData.codigo" type="string">
  Product code/SKU
</ParamField>

<ParamField body="productData.categoria" type="string">
  Product category (e.g., "Helados", "Preparados")
</ParamField>

<ParamField body="productData.estado" type="string">
  Product status (defaults to "Activo")
</ParamField>

<ParamField body="productData.ingredientes" type="string">
  Comma-separated list of ingredients
</ParamField>

<ParamField body="productData.clientes" type="string">
  Comma-separated list of assigned clients
</ParamField>

<ParamField body="productData.fechaVencimiento" type="integer">
  Shelf life in days
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the product was created successfully
</ResponseField>

<ResponseField name="message" type="string" required>
  Confirmation or error message
</ResponseField>

### Example Request

```bash
curl -X POST "https://api.ditzlertotes.com/api/admin/productos" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "action": "create",
    "productData": {
      "nombre": "Helado Frutilla",
      "descripcion": "Helado cremoso de frutilla con trozos de fruta",
      "codigo": "HEL-FRU-001",
      "categoria": "Helados",
      "estado": "Activo",
      "ingredientes": "Leche, Crema, Azúcar, Frutilla, Estabilizantes",
      "clientes": "Retail, Helados",
      "fechaVencimiento": 180
    }
  }'
```

### Example Response

```json
{
  "success": true,
  "message": "Producto creado correctamente"
}
```

### Error Response

```json
{
  "success": false,
  "message": "Ya existe un producto con ese nombre"
}
```

## Update Product

Update an existing product in the catalog.

### Request Body

<ParamField body="action" type="string" required>
  Set to "update"
</ParamField>

<ParamField body="productData" type="object" required>
  Product information object
</ParamField>

### ProductData Fields

<ParamField body="productData.id" type="integer" required>
  Product ID to update
</ParamField>

<ParamField body="productData.nombre" type="string" required>
  Product name
</ParamField>

<ParamField body="productData.descripcion" type="string">
  Product description
</ParamField>

<ParamField body="productData.codigo" type="string">
  Product code/SKU
</ParamField>

<ParamField body="productData.categoria" type="string">
  Product category
</ParamField>

<ParamField body="productData.estado" type="string" required>
  Product status
</ParamField>

<ParamField body="productData.ingredientes" type="string">
  List of ingredients
</ParamField>

<ParamField body="productData.clientes" type="string">
  Assigned clients
</ParamField>

<ParamField body="productData.fechaVencimiento" type="integer">
  Shelf life in days
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the product was updated successfully
</ResponseField>

<ResponseField name="message" type="string" required>
  Confirmation or error message
</ResponseField>

### Example Request

```bash
curl -X POST "https://api.ditzlertotes.com/api/admin/productos" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "action": "update",
    "productData": {
      "id": 1,
      "nombre": "Helado Vainilla Premium Plus",
      "descripcion": "Helado de vainilla premium con vainilla de Madagascar",
      "codigo": "HEL-VAN-001",
      "categoria": "Helados Premium",
      "estado": "Activo",
      "ingredientes": "Leche, Crema, Azúcar, Vainilla de Madagascar",
      "clientes": "Retail, Helados, Preparados",
      "fechaVencimiento": 180
    }
  }'
```

### Example Response

```json
{
  "success": true,
  "message": "Producto actualizado correctamente"
}
```

### Error Responses

```json
{
  "success": false,
  "message": "Ya existe otro producto con ese nombre"
}
```

```json
{
  "success": false,
  "message": "Producto no encontrado"
}
```

## Delete Product

Delete a product from the catalog.

### Request Body

<ParamField body="action" type="string" required>
  Set to "delete"
</ParamField>

<ParamField body="productData" type="object" required>
  Object containing the product ID
</ParamField>

### ProductData Fields

<ParamField body="productData.id" type="integer" required>
  Product ID to delete
</ParamField>

### Response

<ResponseField name="success" type="boolean" required>
  Indicates if the product was deleted successfully
</ResponseField>

<ResponseField name="message" type="string" required>
  Confirmation or error message
</ResponseField>

### Example Request

```bash
curl -X POST "https://api.ditzlertotes.com/api/admin/productos" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "action": "delete",
    "productData": {
      "id": 5
    }
  }'
```

### Example Response

```json
{
  "success": true,
  "message": "Producto eliminado correctamente"
}
```

### Error Response

```json
{
  "success": false,
  "message": "Producto no encontrado"
}
```

## Error Handling

All endpoints return consistent error responses:

### Authentication Error (401)

```json
{
  "success": false,
  "message": "No autorizado"
}
```

### Validation Error (400)

```json
{
  "success": false,
  "message": "Datos de producto requeridos"
}
```

```json
{
  "success": false,
  "message": "ID de producto requerido"
}
```

```json
{
  "success": false,
  "message": "Acción no válida"
}
```

### Server Error (500)

```json
{
  "success": false,
  "message": "Error al conectar con la base de datos."
}
```
