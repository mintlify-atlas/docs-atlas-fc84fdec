<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get DitzlerTotes up and running in minutes</p>
  
  <Steps>
    <Step title="Install Dependencies">
      Clone the repository and install the required packages:
      
      ```bash
      git clone https://github.com/Dicip/DitzlerTotesDB.git
      cd DitzlerTotesDB
      npm install
      ```
    </Step>
    
    <Step title="Configure Environment">
      Create a `.env` file with your database credentials:
      
      ```bash
      DB_USER=your_sql_user
      DB_PASSWORD=your_sql_password
      DB_DATABASE=Ditzler
      DB_SERVER=localhost
      DB_PORT=1433
      PORT=3000
      SESSION_SECRET=your_secret_key_here
      ```
      
      <Tip>Generate a secure session secret using: `node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"`</Tip>
    </Step>
    
    <Step title="Set Up Database">
      Run the database setup script to create tables and stored procedures:
      
      ```sql
      -- Execute in SQL Server Management Studio
      USE master;
      GO
      -- Run the master installation script
      -- Located at: database/00_Master_Install.sql
      ```
    </Step>
    
    <Step title="Start the Server">
      Launch the application:
      
      ```bash
      npm start
      ```
      
      The server will start at `http://localhost:3000`. Default credentials:
      - Email: `admin@ditzler.com`
      - Password: `admin123`
      
      <Warning>Change default credentials in production environments</Warning>
    </Step>
  </Steps>
</div>
